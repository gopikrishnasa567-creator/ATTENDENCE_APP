# 📅 Attendance Management System

A web-based attendance management system built using **Python Flask** and **MySQL**.
Teachers can mark subject-wise attendance and students can view their attendance percentage.

---

## 🚀 Features

- 🔐 Separate login for **Teacher** and **Student**
- 📋 Teacher can mark **Present / Absent** subject-wise
- 📊 Students can view **subject-wise attendance percentage**
- 📅 **Auto date** detection from system
- 🗓️ **Weekly timetable** view for students
- 🎨 Clean and attractive UI

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| Python Flask | Backend Web Framework |
| MySQL | Database |
| HTML/CSS | Frontend |
| Jinja2 | Templating Engine |
